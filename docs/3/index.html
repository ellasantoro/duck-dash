Game Background & Information

Project Features

  • Animates models using nodes from GLTF file
  • Animates object parts based on user key input
  • Functionality based on user key input
  • Rotates using quaternions & euler angles
  • Utilizes raycasters and personally defined event functions to create clicking events based on where you click on the screen, and have 3D objects react to clicking action
  • Sets boundaries using boxes to ensure certain boxes won't intersect
  • Allows sprites to fall off of a platform
  • Allows sprites to jump, be disallowed from double jumping, and change a max jump height based on y position
  • Using flags
  • Connects HTML and JS for score keeping and death screen
  • Uses timers to make objects randomly appear (coins) and makes coins disappear once collided
